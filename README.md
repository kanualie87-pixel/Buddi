BUDDI: AI-Driven Physique Architecture
BUDDI (Bodybuilding User-Driven Dynamic Interface) is a responsive, lightweight web application designed to provide personalized fitness roadmaps. Developed specifically with the UNIMAK student community in mind, it bridges the gap between high-level fitness coaching and local resource availability in Makeni.

🚀 Features
Gender-Specific Profiling: Tailored paths for both Male and Female athletes.

Action-Based Selection: 14 unique fitness objectives (Mass Monster, Bikini Pro, Powerlifting, etc.).

Dynamic Roadmaps: Every plan generates a unique 7-day training split.

Smart Content Injection: Custom nutrition protocols, expert tips, and training guides based on the selected goal.

Localized Advice: Built-in recommendations for affordable protein and energy sources available in local Makeni markets (Cassava, Groundnuts, Fish).

Seamless Navigation: Integrated "Back" buttons for a smooth, recursive user journey.

🛠️ Built With
HTML5: Structural backbone and semantic layout.

CSS3: Custom "Dark Mode" UI, Flexbox/Grid layouts, and interactive hover effects.

JavaScript (ES6+): Logic engine, state management, and dynamic DOM manipulation.

Feather Icons: Professional vector icons for the landing page.

Unsplash API: High-resolution, goal-specific fitness photography.

🖥️ Getting Started
To run the project locally on your Mac or PC:

Clone or Download the project folder.

Ensure you have an active internet connection (to load the Feather Icons and Unsplash images).

Open the index.html file in any modern web browser (Chrome, Safari, or Firefox).

Select your profile and start building your roadmap.
